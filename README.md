# auth

A collection of authentication libraries for PHP. For now it only includes a JWT library.  
  
**Composer install :**  
```
{
    "repositories": [
        {
            "url": "https://github.com/geocata/auth.git",
            "type": "git"
        }
    ],
    "require": {
        "geocata/auth": "~1.0"
    }
}
```
